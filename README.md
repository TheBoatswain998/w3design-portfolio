# W3DESIGN Portfolio

Personal portfolio website for **W3DESIGN**.

This project showcases my design works, including posters, event visuals, avatars and video previews.
The website is built with a clean dark UI, smooth navigation and a focus on visual presentation.

## Features
- Dark modern UI
- Works gallery (images & videos)
- Fullscreen preview for images and videos
- Video previews with autoplay
- Pricing section
- Responsive layout
- Smooth navigation between sections

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
## Preview
You can open the project locally by simply opening `index.html` in your browser.
